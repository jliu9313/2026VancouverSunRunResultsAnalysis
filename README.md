# 2026VancouverSunRunResultsAnalysis
2026 Vancouver Sun Run Results Analysis


An end-to-end Python pipeline that takes the messy fixed-width results dump
from the 2026 Vancouver Sun Run 10K (Sunday, April 19, 2026) and turns it
into a clean dataset, tidy summary tables, and a set of charts.

```
sun-run-2026-analysis/
├── data/
│   ├── raw/SunRun2026Data.txt         # original fixed-width dump
│   └── clean/sunrun2026_clean.csv     # generated by parse_data.py
├── src/
│   ├── parse_data.py                  # regex-based parser
│   ├── analyze.py                     # summary stats + correlations
│   ├── visualize.py                   # matplotlib / seaborn charts
│   └── main.py                        # runs the whole pipeline
├── charts/                            # 9 PNG figures
├── reports/                           # findings.txt + CSV tables
├── requirements.txt
└── README.md
```

## Quick start

```bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
python src/main.py
```

Outputs:

* `data/clean/sunrun2026_clean.csv` — one row per finisher (45,013 rows)
* `reports/findings.txt` — human-readable summary
* `reports/*.csv` — structured tables (by age group, cities, correlations, …)

## Key findings

### 1. Strong correlations

Using Spearman rank correlation on 44,849 rows with full data:

| pair | ρ | interpretation |
| --- | --- | --- |
| `sex_place` ↔ `chip_time_min` | **+0.93** | Trivially true — rank within sex is almost a monotonic function of time. Good sanity check that parsing is correct. |
| `start_delay_sec` ↔ `chip_time_min` | **+0.55** | Runners who start later (further back in the corrals) finish slower. |
| `bib` ↔ `chip_time_min` | **+0.54** | Bib numbers are seeded — low numbers go to faster runners. |
| `age_mid` ↔ `chip_time_min` | +0.19 | Age matters but not as much as the corralling variables. |

See `charts/08_correlation_heatmap.png`.

### 2. The start-line "corral effect" is real — and mostly a selection effect, not a causal one

Bucketing runners by the gap between gun-time and chip-time start:

| start delay | n | median chip time |
| --- | --- | --- |
| 0–30 s | 473 | 37.7 min |
| 2–5 min | 2,822 | 46.5 min |
| 5–10 min | 2,591 | 56.6 min |
| 10–20 min | 3,920 | 59.5 min |
| 20 min + | 20,441 | 66.4 min |

Reading this naively you'd say "starting late makes you slow", but that's the wrong causal direction: **organisers put slower self-estimated runners in later corrals**, so the start-delay variable is downstream of expected pace. When we look *within* the same age × sex cell (see `charts/07_box_by_age.png` and `reports/findings.txt` → "Within-group pace by start delay"), the pace difference between "started in <1 min" and "started 15+ min late" is typically **1.5–2 minutes per km** — that is larger than any plausible pure-crowding penalty (which for a well-marshalled 10K is usually under 30 s/km). So most of the signal is selection; a smaller piece is mechanical crowding in the densest corrals.

### 3. The gender gap widens steadily with age, then collapses at 75+

Median chip time, Women − Men, by age group (minutes):

| age | F − M (min) | F − M (%) |
| --- | --- | --- |
| 19–24 | +10.2 | +17% |
| 30–34 | +10.9 | +18% |
| 40–44 | +13.3 | +21% |
| 50–54 | +15.8 | +24% |
| 60–64 | +19.4 | +27% |
| 65–69 | +24.0 | +31% |
| 75–79 | +8.5 | +9% |
| 80–84 | +7.2 | +8% |

The gap grows from ~17% in the twenties to ~31% in the late 60s, then shrinks sharply in the 75+ cohorts. This is almost certainly a **survivorship / self-selection** effect: very few women race 10Ks at 75+ (n=81 at 75-79 vs. 97 men), and the ones who do are a highly selected athletic subgroup, so their median converges toward the equivalent elderly male group. See `charts/03_gender_gap.png`.

### 4. Performance vs age is a U-shape that bottoms in the late 20s

For men, the fastest median (**58.9 min**) is in the 19–24 bucket; women's fastest median (**68.7 min**) is in 16–18 (n is thin there) with 19–24 a close second. Both sexes slow by roughly **1 min/decade** until age 40, then acceleration of decline kicks in sharply after 60. See `charts/02_median_by_age.png`.

### 5. City-level effects are small — but not zero

Participation is dominated by the Lower Mainland. Median times (limited to cities with ≥100 runners):

| city | n | median (min) |
| --- | --- | --- |
| Victoria | 192 | **61.4** |
| Squamish | 166 | 62.5 |
| North Vancouver | 2,395 | 64.3 |
| Vancouver | 16,969 | 64.9 |
| Port Moody | 602 | 65.4 |
| Pitt Meadows | 147 | 65.8 |
| … | | |
| Surrey | 5,109 | 73.7 |
| Langley | 1,462 | 72.0 |
| Mission | 154 | 77.4 |

Cities closer to the seawall / trail infrastructure (Victoria, North Vancouver, Squamish) skew faster, while car-dependent suburbs (Surrey, Mission, Langley) skew slower. With this many confounders (age mix, income, demographics), this is suggestive rather than conclusive — but it's a clean pattern. See `charts/06_top_cities.png`.

### 6. Bib number is a proxy for expected performance

The `bib` ↔ `chip_time_min` Spearman ρ of +0.54 reflects the race's seeding rule: elites get bibs 1–20, sub-elite wave bibs <5000, then general registration fills the rest. This is not causal — bib numbers don't make you run faster — but it's useful: if you're analysing *any* race and don't know the wave structure, the bib distribution is a free proxy. See `charts/05_bib_vs_time.png`.

## Cause-and-effect summary

| observed correlation | plausible causal story | is it really causal? |
| --- | --- | --- |
| Age → slower times | Physiology (VO₂max, muscle mass) declines with age | **Yes** — well-established biology. |
| Female sex → slower times | Physiology (hemoglobin, VO₂max, muscle fibre type) | **Partly.** Gap among elites ~10%; gap in recreational fields is inflated by self-selection. |
| Late start corral → slower time | Corralling based on predicted pace | **Reverse causal.** Slower runners are *assigned* to later corrals. A minor true-causal piece from crowding exists. |
| Living in distant suburbs → slower time | Less running infrastructure, demographics | **Weakly causal.** Confounded by age mix and opportunity to train. |
| Higher bib number → slower time | Bibs are issued by seeding wave | **Not causal.** Bib is a proxy, not a cause. |

## Data notes / caveats

* The raw file is a page-broken fixed-width dump from sportstats.one. The
  parser in `src/parse_data.py` uses a single regex and does not rely on
  column positions, so it survives the occasional runaway city name.
* Some rows have no Final Place (timed out / finished after tape down).
  These runners still have a chip time and are included in time-based
  analyses.
* Age-group labels `M15-` / `F15-` / `F08` are truncation artefacts of the
  results generator; they are canonicalised to `15-19` / `00-14` in the
  parser.
* `start_delay_sec` can't distinguish "slow corral" from "late arrival at
  the start line" — treat it as an *order-of-start* variable.

## Reproducibility

```bash
python src/parse_data.py      # messy txt -> clean CSV
python src/analyze.py         # clean CSV -> reports/*.csv + findings.txt
python src/visualize.py       # clean CSV -> charts/*.png
```

Everything is deterministic — no random sampling or model fitting with seeds.
